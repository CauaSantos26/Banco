# Sistema Bancário - Java

Aplicação de console que simula operações básicas de uma conta bancária.

## Funcionalidades
- Cadastro de cliente com ou sem saldo inicial
- Depósito e saque com validação de valores

## Como rodar
1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe `Main.java`

## Tecnologias
- Java 17+
